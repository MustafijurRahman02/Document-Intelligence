# AI Document Intelligence Pro

A powerful AI-based document analysis web app built using **Streamlit** and **Azure Document Intelligence**.

## Features

- 📤 Upload Image/PDF or provide URL
- 🤖 Extract text using Azure AI
- 🔊 Text-to-Speech (listen output)
- 🌍 Translate text (multi-language)
- 📄 Download as TXT
- 📥 Export as PDF

---

## Tech Stack

- Python
- Streamlit
- Azure AI Document Intelligence
- gTTS (Text-to-Speech)
- Google Translate API
- ReportLab (PDF generation)
